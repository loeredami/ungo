#Unsual Numbers
are constraints for complex numbers, which buils on the principle, that:
$$
x / 0 \to {[infinity * i, x * i, -infinity * i, -x * i, -infinity, -x, 0, x, infinity]}
$$
As the only thing we know about deviding by 0 is, that if you approach 0 from either side, we approach both infinities in negative and positive space. 

Every value can only exist once, and you use different constraint functions to minimize to one possibility.

Using various math operations like the one above, can increase possbile results.

#Why this exists?
This is not intended for real mathematics, but rather, sci-fi calculations. For example, imagine you have a game that is about quantum mechanics. You COULD do some fancy logic checking to simulate it. OR use these rules to calculate positions, velocities or whatever else you feel like.


For the same reason unsure bools were made, however those can also fit other needs.
